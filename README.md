# discover-leeds-2023

This game made for LEEDS 2023 Game Jam, it's a relaxing game that you paint your art using a paintball gun. Also, the game has screenshot feature to get the art you made as PNG file. Click [here](https://gundulfn.itch.io/discover) to download game for Windows/macOS.
